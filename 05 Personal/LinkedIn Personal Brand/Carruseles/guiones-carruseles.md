## Carrusel 2: "De idea a proyecto en 5 pasos"

**Slide 1 — Portada**
> DE IDEA A PROYECTO
> Mi proceso en 5 pasos

**Slide 2 — Paso 1: Define el problema**
> 1. DEFINE EL PROBLEMA
> No la solución.
> • ¿Qué duele?
> • ¿Qué podrías hacer 10x más fácil?
> • ¿Qué te apasiona resolver?

**Slide 3 — Paso 2: Prototipo en 24h**
> 2. PROTOTIPO EN 24 HORAS
> Sin perfeccionismo.
> • Mínimo producto funcional
> • Prueba con usuarios reales
> • Falla rápido, aprende rápido

**Slide 4 — Paso 3: Iteración guiada**
> 3. ITERACIÓN GUIADA
> Por feedback, no por suposiciones.
> • Mide uso real
> • Pregunta a tus usuarios
> • Una mejora a la vez

**Slide 5 — Paso 4: Automatiza lo repetitivo**
> 4. AUTOMATIZA
> Lo que no aporta valor diferencial.
> • Testing
> • Deploy
> • Documentación
> • Reportes

**Slide 6 — Paso 5: Comparte el proceso**

## Carrusel 3: "5 usos prácticos de IA para devs"

**Slide 1 — Portada**
> 5 USOS DE IA PARA DEVS
> No es teoría, es práctica

**Slide 2 — 1. Code Review Automatizado**
> 1. CODE REVIEW CON IA
> • Detecta bugs antes del merge
> • Sugiere mejoras de rendimiento
> • Revisa consistencia con estándares
> 
> Prompt: "Revisa este PR y encuentra vulnerabilidades"

**Slide 3 — 2. Tests Automáticos**
> 2. GENERACIÓN DE TESTS
> • A partir del código fuente
> • Cobertura de casos borde
> • Tests unitarios + integración
> 
> Prompt: "Genera tests unitarios para esta función"

**Slide 4 — 3. Documentación**
> 3. DOCUMENTACIÓN AUTOMÁTICA
> • Comentarios de código
> • README profesional
> • Documentación de API
> 
> Prompt: "Escribe documentación técnica para este endpoint"

**Slide 5 — 4. Refactorización**
> 4. REFACTORIZACIÓN ASISTIDA
> • Identifica code smells
> • Sugiere patrones de diseño
> • Optimiza rendimiento
> 
> Prompt: "Refactoriza este código siguiendo principios SOLID"

**Slide 6 — 5. Debugging Acelerado**
> 5. DEBUGGING CON IA
> • Analiza logs
> • Sugiere causas raíz
> • Propone soluciones
> 
> Prompt: "Este es el error: [log]. ¿Cuál es la causa probable?"

**Slide 7 — Guards & Comparte**
> GUARDA ESTE CARRUSEL
> La próxima vez que programes, prueba estos prompts.
> 
> Sígueme para más contenido de ingeniería.
