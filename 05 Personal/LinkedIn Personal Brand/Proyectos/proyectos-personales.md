# Proyectos Personales

## Proyectos destacados

### 1. Sistema de Segundo Cerebro en Obsidian
**Tipo:** Productividad / Conocimiento
**Stack:** Obsidian, Dataview, Git
**Descripción:** Sistema completo de gestión de conocimiento personal con 1,200+ notas interconectadas, MOCs (Maps of Content), dashboards de productividad y automatización de categorización.
**Logro:** Reducción del 60% en tiempo de búsqueda de información.

### 2. Asistente de Automatización Personal
**Tipo:** Automatización
**Stack:** Python, APIs, Cron
**Descripción:** Suite de scripts que automatizan tareas diarias: organización de archivos, respaldo de datos, reportes automáticos y recordatorios inteligentes.
**Logro:** Automatización del 80% de tareas administrativas digitales.

### 3. Web App para Gestión de Proyectos Personales
**Tipo:** Desarrollo Web
**Stack:** React, Node.js, MongoDB
**Descripción:** Aplicación web ligera para gestionar proyectos personales, con Kanban integrado, temporizador Pomodoro y estadísticas de productividad.

### 4. Framework de Prompt Engineering
**Tipo:** IA / Herramienta
**Stack:** Documentación técnica, ejemplos prácticos
**Descripción:** Framework personal de prompt engineering para optimizar interacciones con modelos de lenguaje. Incluye plantillas, ejemplos y guías por caso de uso.

## Mi enfoque de proyectos personales

> "Construye lo que te gustaría que existiera."

Cada proyecto personal resuelve un problema que yo mismo tenía. No pretendo competir con productos establecidos. Son herramientas que necesitaba y que comparto por si alguien más las encuentra útiles.

### Criterios para empezar un proyecto

1. **¿Resuelve un problema real para mí?** — Si no, no lo construyo.
2. **¿Puedo tener un prototipo en <1 semana?** — Si es más largo, lo divido.
3. **¿Aprenderé algo nuevo?** — Cada proyecto debe enseñarme algo.
4. **¿Podría compartirlo?** — Si no puedo mostrarlo, quizá no vale la pena.
