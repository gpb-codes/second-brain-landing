# Keyboard Shortcuts - Windows

## System
| Shortcut | Action |
|----------|--------|
| `Win+D` | Show desktop |
| `Win+E` | File Explorer |
| `Win+L` | Lock screen |
| `Win+Tab` | Task view |
| `Alt+Tab` | Switch windows |

## Browser
| Shortcut | Action |
|----------|--------|
| `Ctrl+T` | New tab |
| `Ctrl+W` | Close tab |
| `Ctrl+Shift+T` | Reopen closed tab |
| `Ctrl+L` | Focus address bar |
| `F5` | Refresh |

## VS Code
| Shortcut | Action |
|----------|--------|
| `Ctrl+P` | Quick open |
| `Ctrl+Shift+P` | Command palette |
| `Ctrl+`` | Toggle terminal |
| `Ctrl+Shift+F` | Search in files |
| `Alt+↑/↓` | Move line |

## Productivity
| Shortcut | Action |
|----------|--------|
| `Ctrl+C/V/X` | Copy/Paste/Cut |
| `Ctrl+Z/Y` | Undo/Redo |
| `Ctrl+F` | Find |
| `Ctrl+S` | Save |
