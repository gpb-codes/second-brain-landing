## Calendario de crossposting

Cuando publiques en una plataforma, adapta a las otras:

```
PUBLICACION PRINCIPAL (donde se origina)
  └── Linkedin (texto)
  └── Instagram (visual)
  └── GitHub (codigo)
  └── YouTube (video)

Ejemplo:
1. Grabas video para YouTube (Miercoles)
2. Lunes siguiente: Post en LinkedIn con insights del video
3. Martes: Reel en Instagram con clip destacado
4. Miercoles: Subes el video a YouTube
5. GitHub: README actualizado con link al video
```

## Consistency checklist semanal

- [ ] Misma foto de perfil en las 4 plataformas
- [ ] Bio actualizada con misma informacion
- [ ] Mismos proyectos destacados en todas
- [ ] Links a las otras plataformas visibles
- [ ] Publicacion principal replicada (adaptada) en 2+ plataformas
- [ ] Sin informacion contradictoria entre plataformas
