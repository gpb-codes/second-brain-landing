# Glossario Universal del Second Brain

> [!info] Términos compartidos
> Definiciones que aplican a múltiples áreas del vault PACE.

---

## A

**API Key** → Clave de acceso a servicios. En agentes: para LLMs. En acceso remoto: para VPNs y servicios.

**Automatización** → Proceso de ejecutar tareas sin intervención humana. Común en agentes IA, RDP, y contenido.

---

## C

**Container (Docker)** → Empaquetado de software. En agentes: para sandbox. En acceso remoto: para servidores.

---

## G

**GitHub** → Plataforma de código. Repos documentados, perfil profesional, auto-update del vault.

---

## L

**LLM** → Large Language Model. Modelos de lenguaje como GPT, Claude, Mistral.

---

## M

**MCP** → Model Context Protocol. Estándar para integración de herramientas en agentes.

---

## O

**Ollama** → Runtime local para ejecutar modelos LLM sin costo de API.

---

## P

**PACE** → Proyectos, Áreas, Conexiones, Extracciones. Metodología de organización del segundo cerebro.

---

## R

**RDP** → Remote Desktop Protocol. Protocolo de Microsoft para acceso remoto a Windows.

---

## S

**Sandbox** → Ambiente aislado para ejecutar código peligroso. Docker, VMs.

**Second Brain** → Sistema de gestión de conocimiento personal. Segunda memoria externa.

---

## T

**Tailscale** → VPN basada en WireGuard. Acceso seguro a dispositivos remotos.

---

## V

**Vault** → Carpeta de Obsidian que contiene notas organizadas por tema.

**VPN** → Virtual Private Network. Red privada sobre internet público.
