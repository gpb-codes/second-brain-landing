# Carruseles para Instagram

## Diferencias con LinkedIn

| Aspecto | LinkedIn | Instagram |
|---------|----------|-----------|
| **Formato** | PDF en documento | Imágenes 1:1 o 4:5 |
| **Texto** | Más texto por slide | Menos texto, más visual |
| **Estilo** | Minimalista profesional | Colorido, llamativo |
| **Cover** | Título en portada | Portada que pare feed |
| **Slides** | 7-12 | 5-10 |

---

## 15 ideas de carruseles para Instagram

### Educativos (5)

1. **"IA explicada en 5 slides"**
   - Slide 1: ¿Qué es IA? (ilustración simple)
   - Slide 2: Tipos de IA (infografía)
   - Slide 3: Cómo funciona un LLM (diagrama)
   - Slide 4: Herramientas IA para empezar
   - Slide 5: CTA: guarda para aprender

2. **"Mi stack tecnológico 2026"**
   - Slide 1: Portada llamativa
   - Slide 2-6: Cada herramienta con icono + 1 línea
   - Slide 7: Costos y alternativas
   - Slide 8: Template para armar tu stack

3. **"5 pasos para construir un proyecto tech"**
   - Slide 1: Portada
   - Slide 2-6: Cada paso con icono + descripción
   - Slide 7: Checklist descargable

4. **"10 comandos de terminal que te harán lucir como pro"**
   - Slide 1: Portada misteriosa
   - Slide 2-11: Cada comando con screenshot
   - Slide 12: Bonus comando secreto

5. **"Cómo escribir prompts efectivos"**
   - Slide 1: Portada
   - Slide 2-4: La estructura del prompt
   - Slide 5-7: Ejemplos buenos vs malos
   - Slide 8: Plantilla descargable

### Inspiracionales (5)

6. **"Mi viaje como programador"** — Timeline visual con fotos
7. **"Errores que todo dev comete"** — Cards con cada error + solución
8. **"De 0 a 1000 horas de código"** — Progresión visual
9. **"Mentiras que te dices cuando empiezas"** — Frases + realidad
10. **"Hábitos que cambiaron mi productividad"** — Antes/Después visual

### Detrás de Escena (5)

11. **"Así funciona Drakkar Labs por dentro"** — Tour visual
12. **"Mi semana como founder técnico"** — Timeline de lunes a domingo
13. **"Las herramientas que uso para todo"** — Grid visual
14. **"Mi setup de programación"** — Fotos etiquetadas
15. **"Un día en mi vida tech"** — Horario visual con fotos

## Plantillas de diseño para slides

### Plantilla educativa
```
┌─────────────────────────┐
│  TÍTULO (negrita grande)│
│                         │
│  [Icono grande]         │
│                         │
│  Descripción breve      │
│  (2 líneas máx)         │
│                         │
│  #drakkarlabs           │
└─────────────────────────┘
```

### Plantilla lista
```
┌─────────────────────────┐
│  1/5                    │
│                         │
│  [Número grande]        │
│  Título del item        │
│  ──────────────         │
│  Descripción corta      │
│                         │
└─────────────────────────┘
```

### Plantilla comparativa
```
┌────────────┬────────────┐
│   ANTES    │   DESPUÉS  │
│            │            │
│ [Imagen A] │ [Imagen B] │
│            │            │
│ Texto A    │ Texto B    │
└────────────┴────────────┘
```

## Checklist de diseño

| Elemento | ✓ | Notas |
|----------|---|-------|
| Portada que pare en el feed | ☐ | Texto grande, llamativo |
| Misma tipografía en todo | ☐ | Sans-serif legible |
| Colores de marca consistentes | ☐ | Azul + Púrpura + Blanco |
| Iconos simples (Noun Project o básicos) | ☐ | Sin saturar |
| Texto legible en móvil | ☐ | Mínimo 24px |
| CTA claro al final | ☐ | Guarda, comparte, comenta |
| Logo/marca en cada slide | ☐ | Esquina inferior |
| Slice final con resumen | ☐ | Recapitulación |
