# Estructura de Slides para Carruseles

## Plantilla universal de carrusel

| Slide | Contenido | Elementos clave | Longitud |
|-------|-----------|-----------------|----------|
| 1 | **Portada** | Título grande, subtítulo, tu nombre | 3-5 palabras título |
| 2-6 | **Contenido** | 1 idea por slide, visual + texto mínimo | Máximo 20 palabras/slide |
| 7 | **Bonus/Resumen** | Recapitulación o insight adicional | 3-4 puntos |
| 8 | **CTA** | Llamada a la acción, perfil, hashtag | 1-2 frases |

---

## Fórmulas de estructura

### Fórmula 1: Lista
```
1. Portada ("X cosas que...")
2. Item 1
3. Item 2
4. Item 3
5. Item 4
6. Item 5
7. Bonus (item extra)
8. CTA
```

**Mejor para:** Consejos, herramientas, aprendizajes

### Fórmula 2: Proceso
```
1. Portada ("Mi proceso para...")
2. Paso 1
3. Paso 2
4. Paso 3
5. Paso 4
6. Paso 5
7. Resultado final
8. CTA + template
```

**Mejor para:** Tutoriales, metodologías, frameworks

### Fórmula 3: Antes/Después
```
1. Portada ("Antes y después de...")
2. El problema (antes)
3. Por qué no funcionaba
4. La solución (después)
5. Cómo llegué
6. Resultados
7. Lección aprendida
8. CTA
```

**Mejor para:** Casos de estudio, transformaciones

### Fórmula 4: Comparativa
```
1. Portada ("X vs Y: ¿Cuál elegir?")
2. Opción A: ventajas
3. Opción A: desventajas
4. Opción B: ventajas
5. Opción B: desventajas
6. Comparativa directa (tabla)
7. Mi recomendación
8. CTA
```

**Mejor para:** Herramientas, tecnologías, stacks

## Diseño visual de slides

| Elemento | Recomendación |
|----------|---------------|
| Fondo | Gradiente oscuro o color sólido con contraste |
| Tipografía | Sans-serif, negrita para títulos |
| Colores | 2-3 colores máximo + 1 acento |
| Imágenes | Iconos simples o capturas de pantalla |
| Logo/Marca | Esquina inferior derecha |
| Número de slide | Esquina inferior izquierda (opcional) |

## Herramientas para crear carruseles

- **Canva** — Plantillas prediseñadas
- **Figma** — Control total del diseño
- **PowerPoint** — Exportar a PDF
- **Google Slides** — Gratuito y colaborativo
- **Gamma.app** — IA para generar presentaciones
