# Ideas para Carruseles en LinkedIn

## Formato carrusel: Cuándo usarlo

El carrusel (documento PDF de varias slides) funciona mejor para:
- Contenido educativo paso a paso
- Listas y frameworks
- Comparativas
- Guías prácticas
- Storytelling visual

---

## 10 ideas de carruseles técnicos

### 1. "Mi stack de IA en 2026"
- Slide 1: Portada con el stack completo
- Slide 2-5: Cada herramienta con screenshot
- Slide 6: Costos y alternativas
- Slide 7: Plantilla para copiar

### 2. "De idea a proyecto: mi proceso en 5 pasos"
- Slide 1: El problema que quería resolver
- Slide 2-6: Cada paso con diagrama
- Slide 7: Resultado final
- Slide 8: Template descargable

### 3. "Errores que cometí como dev (y cómo evitarlos)"
- Slide 1: Portada provocadora
- Slide 2-6: Cada error + solución
- Slide 7: Lección principal

### 4. "IA para devs: 5 usos prácticos"
- Slide 1: Preview de los 5 usos
- Slide 2-6: Cada uso con ejemplo código
- Slide 7: Prompt templates

### 5. "Mi setup de productividad técnica"
- Slide 1: Vista general del setup
- Slide 2-5: Cada herramienta con descripción
- Slide 6: Costo total
- Slide 7: Enlaces de descarga

### 6. "Cómo organicé 1200 notas en Obsidian"
- Slide 1: El caos inicial (screenshot)
- Slide 2-5: El sistema de organización
- Slide 6: Antes vs Después
- Slide 7: Tips para empezar

### 7. "Guía para empezar con automatización"
- Slide 1: ¿Por qué automatizar?
- Slide 2-4: Herramientas gratuitas
- Slide 5-7: Ejemplos prácticos
- Slide 8: Roadmap de aprendizaje

### 8. "10 herramientas open source que uso"
- Slide 1: Portada
- Slide 2-11: Cada herramienta (1 slide)
- Slide 12: Bonus extra

### 9. "El framework que uso para decidir qué construir"
- Slide 1: El problema de decidir
- Slide 2-4: Mi framework (matriz)
- Slide 5-7: Ejemplos aplicados
- Slide 8: Plantilla en blanco

### 10. "Métricas que importan al construir un proyecto tech"
- Slide 1: No todo son estrellas en GitHub
- Slide 2-7: Cada métrica explicada
- Slide 8: Dashboard recomendado

## Tips para carruseles virales

| Factor | Cómo aplicarlo |
|--------|---------------|
| Portada impactante | Título grande, promesa clara, sin saturar |
| 7-12 slides | Rango óptimo de engagement |
| Slide final con CTA | "Guarda esto", "Comparte", "Sígueme" |
| Texto mínimo | La imagen vende, el texto complementa |
| Marca consistente | Misma paleta, tipografía, estilo |
| Sin fondos blancos aburridos | Usa gradientes o colores de marca |
