## Que es PACE

<div align="center" style="background: linear-gradient(135deg, #0F172A 0%, #1E293B 100%); border-radius: 20px; padding: 32px; border: 1px solid #22C55E;">

Construyo un sistema de **conocimiento personal** donde la **estructura es intencional**, las **conexiones son automaticas** y la **organizacion elimina la friccion**.

Mi vault se organiza con la metodologia **PACE**: **P**royectos, **A**reas, **C**onexiones, **E**xtracciones.

## Estructura del Vault

<div align="center" style="background: #0F172A; border-radius: 20px; padding: 32px; border: 1px solid #22C55E; box-shadow: 0 0 40px rgba(34, 197, 94, 0.1);">

```
Second-Brain (Boveda PACE)
|
+-- 00 Inbox                  --> Captura rapida
+-- 01 Proyectos (P)          --> Iniciativas con deadline
+-- 02 Areas (A)              --> Responsabilidades continuas
+-- 03 Conexiones - MOCs (C)  --> Mapas de conocimiento
+-- 04 Extracciones AI (E)    --> Logs y destilados
+-- 05 Archivo                --> Contenido historico
+-- 99 Sistema                --> Configuracion y scripts
```
