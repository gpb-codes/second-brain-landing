# Plantillas Reutilizables para Publicaciones

## Plantilla 1: Opinión Técnica

```
[Hook provocador — 1-2 líneas]

—

[Contexto — 2-3 líneas explicando el tema]

—

[Argumento principal — 3-5 líneas con tu postura]

—

[Ejemplo personal o datos — 2-3 líneas]

—

[Conclusión y reflexión final — 2-3 líneas]

—

[CTA — pregunta abierta]

—

[Hashtags]
```

## Checklist pre-publicación

| Elemento | ✓ |
|----------|---|
| Hook fuerte en primeras 3 líneas | ☐ |
| Contenido alinea con pilar de contenido | ☐ |
| Aporta valor real (no ruido) | ☐ |
| CTA claro al final | ☐ |
| 5-6 hashtags relevantes | ☐ |
| Sin errores ortográficos | ☐ |
| Formato legible (párrafos cortos) | ☐ |
| Sin links externos en texto principal (en comentarios) | ☐ |
| Imagen adjunta si aplica | ☐ |
| Responderé a los comentarios | ☐ |
