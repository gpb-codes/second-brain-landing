# Plantillas Visuales para Instagram

## Plantilla 1: Cita técnica

```
┌─────────────────────────────┐
│                             │
│    "EL CÓDIGO ES POESÍA     │
│     QUE EL COMPILADOR       │
│     ENTIENDE"               │
│                             │
│    — Gabriel Pedreros       │
│                             │
│    #drakkarlabs             │
└─────────────────────────────┘
```

**Fondo:** Azul oscuro degradado
**Tipografía:** Sans-serif negrita, blanco
**Uso:** Historias, posts de reflexión

## Checklist de publicación

- [ ] Dimensiones correctas (1080x1080 o 1080x1350)
- [ ] Texto legible sin hacer zoom
- [ ] Logo/marca visible
- [ ] CTA claro (guardar, compartir, comentar)
- [ ] Descripción con copy + hashtags
- [ ] 3-5 hashtags en descripción
- [ ] 15-20 hashtags en primer comentario
- [ ] Sin enlaces externos en el post (solo en bio)
- [ ] Responder a comentarios en 1h
