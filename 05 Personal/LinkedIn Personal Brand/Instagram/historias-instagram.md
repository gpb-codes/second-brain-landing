# Historias para Instagram

## Estrategia de historias

Las historias son para contenido efímero, detrás de escena, y engagement rápido. No necesitan ser perfectas. Necesitan ser **auténticas**.

| Formato | Frecuencia | Objetivo |
|---------|------------|----------|
| Foto del setup | 2-3x/semana | Conexión visual |
| Poll sobre tecnología | 1-2x/semana | Engagement |
| Preguntas (AMA) | 1x/semana | Comunidad |
| Detrás de escena | 2x/semana | Autenticidad |
| Re-post de feed | 1x/semana | Recordatorio |
| Clip rápido de código | 1-2x/semana | Valor técnico |

## Plantillas visuales para stories

### Plantilla "Tip rápido"
```
Fondo: Azul oscuro
Texto grande: [EL TIP]
Subtítulo: "Tip del día"
Esquina: Logo de Drakkar Labs
```

### Plantilla "Detrás de escena"
```
Fondo: Foto del momento
Texto: Contexto de 1 línea
Tag: @drakkarlabs
```

### Plantilla "Pregunta"
```
Fondo: Degradado púrpura
Texto: "[Pregunta]?"
Sticker de preguntas superpuesto
```

### Plantilla "Resultado"
```
Fondo: Antes (izq) vs Después (der)
Texto: "Antes: [X]  →  Después: [Y]"
```
