# AWS - Que es Amazon Web Services

> [!info] Descripcion
> Amazon Web Services (AWS) es la plataforma de computacion en la nube mas completa y ampliamente adoptada del mundo, ofreciendo mas de 200 servicios desde centros de datos globales.

---

## Que es AWS

AWS es una plataforma de servicios en la nube que proporciona:
- **Computacion**: EC2, Lambda, ECS
- **Almacenamiento**: S3, EBS, EFS
- **Bases de datos**: RDS, DynamoDB, Aurora
- **Redes**: VPC, Route 53, CloudFront
- **Seguridad**: IAM, KMS, WAF
- **IA/ML**: SageMaker, Rekognition, Comprehend

## Historia y Evolucion

- **2006**: Lanzamiento de S3 y EC2
- **2010**: AWS supera los $1B en ingresos anuales
- **2015**: AWS se convierte en el lider del mercado cloud
- **2020+**: Expansion en servicios de IA, ML y edge computing

## Ventajas de AWS

### Beneficios Principales
- **Paga por uso**: Sin inversion inicial en infraestructura
- **Escalabilidad**: Crece o decrece segun demanda
- **Velocidad**: Despliegue en minutos, no meses
- **Global**: 31 regiones, 99 zones de disponibilidad
- **Seguridad**: Cumplimiento con mas de 90 estandares

### Modelo de Responsabilidad Compartida
- **AWS**: Seguridad OF the cloud (infraestructura fisica)
- **Cliente**: Seguridad IN the cloud (datos, configuracion)

## Servicios Principales

### Nivel Gratuito (Always Free)
- EC2: 750 horas/mes (t2.micro)
- S3: 5 GB storage
- Lambda: 1M requests/mes
- DynamoDB: 25 GB
- RDS: 750 horas/mes (db.t2.micro)

### Servicios Mas Utilizados
1. **EC2**: Maquinas virtuales
2. **S3**: Almacenamiento de objetos
3. **Lambda**: Computacion serverless
4. **RDS**: Bases de datos relacionales
5. **VPC**: Redes privadas virtuales

## Casos de Uso

### Startups
- Costos bajos para comenzar
- Escalabilidad automatica
- Global desde el dia 1

### Empresas
- Migracion de workloads
- Disaster recovery
- Cumplimiento regulatorio

### Gobierno
- Seguridad avanzada
- Cumplimiento FedRAMP
- Soporte gouvernamental

---

## Aplicacion

### Donde Usar
- Desarrollo de aplicaciones web y moviles
- Big Data y analitica
- Internet of Things (IoT)
- Machine Learning e Inteligencia Artificial
- Juegos en la nube
- Archivos multimedia y streaming

### Precauciones
- Costos inesperados por recursos olvidados
- Complejidad en la seleccion de servicios
- Requiere conocimiento de seguridad en la nube
- Vendor lock-in si no se diseña correctamente
