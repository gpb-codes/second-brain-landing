# CV / Hoja de Vida — Adaptado a cada plataforma

## LinkedIn

### Formato del resumen ejecutivo
```
Estudiante de Ingeniería Informática con proyectos reales en inteligencia
artificial, automatización de procesos y desarrollo web. Fundador de
Drakkar Labs, laboratorio de proyectos tecnológicos donde aplico IA,
automatización y productividad técnica para resolver problemas concretos.

Areas de enfoque:
- Integracion de APIs LLM (OpenAI, Claude, Gemini)
- Automatizacion de flujos de trabajo con Python
- Desarrollo de asistentes inteligentes y agentes IA
- Sistemas de conocimiento personal (Obsidian PKM)
```

### Estructura de experiencia en LinkedIn
| Seccion | Contenido |
|---------|-----------|
| **Titular** | Estudiante de Ingenieria Informatica | IA & Automatizacion | Founder @ Drakkar Labs |
| **Acerca de** | Resumen ejecutivo + filosofia + que busco |
| **Experiencia** | Drakkar Labs (actual), proyectos destacados |
| **Formacion** | Ingenieria Informatica |
| **Licencias** | Certificaciones relevantes |

### Keywords para ATS (Applicant Tracking System)
- Inteligencia Artificial, Machine Learning, Automatizacion
- Desarrollo Web, Full-Stack, Python, JavaScript, TypeScript
- Arquitectura de Software, APIs, Sistemas Distribuidos
- Prompt Engineering, LLMs, AI Agents
- Productividad, Obsidian, Open Source

## Instagram

### Formato: Highlights de carrera
Usa los highlights de Instagram para contar tu historia profesional.

**Highlight: Trayectoria**
- Slide 1: "Ingeniero Informatico - [Universidad]"
- Slide 2: "Primer trabajo en [area]"
- Slide 3: "Descubri la IA y la automatizacion"
- Slide 4: "Funde Drakkar Labs"
- Slide 5: "Hoy: construyendo el futuro"

**Formato visual para cada slide:**
```
Fondo: Foto profesional o gradiente azul
Texto: 1 linea de logro
Icono: Relacionado (codigo, IA, proyecto)
```

### Publicaciones de CV visual
- "Mi trayectoria en 5 slides" (carrusel)
- "Un dia como ingeniero informatico" (Reel)
- "Herramientas que uso a diario" (post del feed)
