# Reels para Instagram — Ideas y Guiones

## Por qué Reels importa

Los Reels son el formato con mayor alcance orgánico en Instagram. Para un perfil técnico, son la mejor forma de mostrar personalidad y conocimiento en segundos.

---

## Estructura de un Reel técnico

### Fórmula ganadora (15-30s)

```
0:00-0:03 → Hook visual fuerte (texto grande en pantalla)
0:03-0:10 → Desarrollo del tema (valor rápido)
0:10-0:20 → Clímax o revelación
0:20-0:25 → Resultado o aprendizaje
0:25-0:30 → CTA + ¿Sígueme para más?
```

### Ejemplo guion: "Automaticé mi mañana con Python"

```
[0:00] Texto: "Automaticé mi mañana con Python 🐍"
[0:03] Video: Timelapse de escribiendo código
[0:08] Texto: "Antes: 20 min haciendo esto manual"
[0:12] Video: Clip de script ejecutándose
[0:18] Texto: "Ahora: 0 min. Lo hace solo."
[0:22] Video: Día normal, café de por medio
[0:26] Texto: "¿Tú qué automatizarías?"
[0:28] CTA: "Sígueme para más automatización 🤖"
```

## Tips técnicos para Reels

| Aspecto | Recomendación |
|---------|--------------|
| Duración | 15-30s (mejor retención) |
| Música | Trends del momento |
| Texto | Siemnea con subtítulos |
| Aspect ratio | 9:16 vertical |
| Cover | Frame con texto llamativo |
| Velocidad | Edición rápida, sin pausas |
| Hashtags | 3-5 en descripción + 5-10 en comentarios |

---

## Equipamiento mínimo

- **Cámara:** Smartphone (cámara trasera)
- **Micrófono:** El del teléfono o uno externo básico
- **Edición:** CapCut (gratuito) o Premiere Rush
- **Captura de pantalla:** OBS Studio (gratuito)
- **Mini trípode:** Cualquiera de 10-15€
