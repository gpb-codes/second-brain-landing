## GitHub — El portafolio tecnico definitivo

### Organizacion de repositorios

| Tipo | Repos | Proposito |
|------|-------|-----------|
| **Proyectos flagship** | 2-3 | Lo mejor que has construido |
| **Herramientas** | 3-5 | Utilidades reutilizables |
| **Templates** | 2-3 | Punto de partida para otros |
| **Experimentos** | 3-5 | Proyectos de aprendizaje |
| **Archivados** | Los que ya no mantienes |

### README de portafolio

```markdown
# Portafolio Tecnico

Bienvenido a mi portafolio de proyectos. Aqui encuentras
el codigo de todo lo que construyo.

## Estructura

| Categoria | Repos | Descripcion |
|-----------|-------|-------------|
| IA | 4 | Asistentes, agentes, pipelines |
| Web | 3 | Aplicaciones full-stack |
| Productividad | 2 | Templates Obsidian, scripts |
| Experimentos | 5 | Prototipos y pruebas |

## Proyecto destacado

[AI Automation Toolkit]
Automatiza tareas con IA usando Python y APIs de LLMs.
Features: transcripcion, resumen, documentacion automatica.

## En progreso

- [Framework de Agentes IA] - Memoria persistente
- [Dev Dashboard] - Metricas de productividad

## Stats

[GitHub Stats]
[Top Languages]
```

### Caso de estudio en README

```markdown

## Solucion

## Resultados

- 90% menos errores
- 100% de cobertura

## Arquitectura
[Diagrama]

## Stack
Python, OpenAI, Whisper, Notion API, GitHub API
```

## Portafolio web (opcional)

Si decides tener sitio propio:

### Secciones recomendadas
1. **Inicio** — Quien eres + lo que haces
2. **Proyectos** — Cada proyecto con descripcion + captura
3. **Blog** — Articulos tecnicos
4. **Contacto** — Formulario + redes

### Stack recomendado
- **Next.js** o **Astro** (static site)
- **Deploy**: Vercel o Netlify (gratis)
- **Dominio**: gpedreros.dev o gabriel.dev
