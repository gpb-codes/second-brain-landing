## 🧠 Métodos

### PARA Method
- **P**rojects: Con plazo definido
- **A**reas: Responsabilidades continuas
- **R**esources: Temas de interés
- **A**rchives: Completado o inactivo

### Zettelkasten
1. Notas literales (citas)
2. Notas de reinterpretación
3. Notas de conexión
4. Notas atómicas (1 idea por nota)
