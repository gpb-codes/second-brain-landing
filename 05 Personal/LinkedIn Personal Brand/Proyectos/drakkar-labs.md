# Drakkar Labs — Laboratorio de Proyectos Tecnológicos

## ¿Qué es Drakkar Labs?

Drakkar Labs es mi laboratorio personal de proyectos tecnológicos. Un espacio donde exploro la intersección entre inteligencia artificial, automatización, desarrollo web y productividad técnica.

No es una startup tradicional ni una consultora. Es un **entorno de experimentación** donde las ideas se convierten en prototipos funcionales.

---

## Proyectos activos

### 1. Asistente IA para Obsidian
**Estado:** En desarrollo
**Stack:** Python, Claude API, Obsidian Plugin API
**Descripción:** Asistente inteligente que organiza notas automáticamente, sugiere conexiones entre ideas y mantiene el sistema de conocimiento actualizado.

### 2. Pipeline de Automatización de Reuniones
**Estado:** Activo
**Stack:** Whisper, GPT-4, Python, Notion API, GitHub API
**Descripción:** Pipeline completo que transcribe reuniones, extrae decisiones, documenta en Notion y crea tareas en GitHub.

### 3. Plataforma de Agentes IA
**Estado:** En planificación
**Stack:** Python, LangChain, APIs LLM
**Descripción:** Framework para crear agentes IA autónomos con memoria persistente, herramientas externas y ejecución multi-paso.

### 4. Herramientas de Productividad Técnica
**Estado:** Iteración constante
**Stack:** JavaScript/TypeScript, Node.js
**Descripción:** Colección de herramientas ligeras para optimizar flujos de trabajo técnicos.

## Filosofía del laboratorio

| Principio | Descripción |
|-----------|-------------|
| Construir rápido | Prototipo funcional antes que perfección |
| Compartir el proceso | Aprender en público, enseñar mientras aprendo |
| Fracasar barato | Fallar rápido, aprender, pivotear |
| Open source primero | Lo que pueda compartirse, se comparte |
| Problemas reales | No construir soluciones buscando problema |

---

## ¿Por qué "Drakkar"?

Drakkar evoca exploración, navegación en territorios desconocidos, construcción artesanal y viajes largos. Exactamente lo que representa este laboratorio: explorar aguas tecnológicas desconocidas, construir con método y compartir el mapa con quien quiera seguirlo.
