# Post 02 — Cómo construí un asistente IA en un fin de semana

**Formato:** Caso práctico / Tutorial
**Tono:** Educativo, cercano
**Pilar:** Proyectos / IA

---

El mes pasado me propuse un reto:

Construir un asistente IA funcional en **un fin de semana**.

No para venderlo.
No para impresionar.
Para demostrarme que se podía.

Este fue el proceso:

—

### Sábado 9:00 AM — Definir el problema

En lugar de construir "algo con IA", definí un problema real:
> Automatizar la organización de mis notas en Obsidian.

Cada semana pierdo tiempo categorizando y linkeando notas.
Quería un asistente que lo hiciera por mí.

—

### Sábado 2:00 PM — Arquitectura

- API de Claude para procesamiento de texto
- Script en Python para leer/ordenar archivos .md
- Lógica de categorización basada en embeddings
- Output: notas organizadas con enlaces sugeridos

—

### Domingo 6:00 PM — Primer prototipo

No era perfecto.
Algunas categorizaciones fallaban.
Pero el 80% de las notas quedaban organizadas automáticamente.

El 20% restante? Lo reviso yo en 5 minutos.

—

### La lección

No necesitas un equipo, ni presupuesto, ni meses.

Necesitas:
1. Un problema real
2. Conocimiento técnico básico
3. APIs accesibles
4. Ganas de experimentar

—

Hoy ese asistente sigue funcionando.
Ahorrándome ~2 horas semanales.
Y me dio ideas para 3 proyectos más.

—

¿Qué problema pequeño podrías resolver este fin de semana con IA?
