# Compatible - Integraciones

## Asistentes de IA

### ChatGPT
- **Uso:** Buscar notas, resumir contenido, generar ideas
- **Como conecta:** Via API o copiar/pegar contexto
- **Ejemplo:** "Resumeme las notas de mi vault de DevOps"

### Claude
- **Uso:** Analisis profundo, generacion de contenido, investigacion
- **Como conecta:** Via API o interfaz web
- **Ejemplo:** "Crea un resumen ejecutivo de mis objetivos 2026"

### Gemini
- **Uso:** Busqueda rapida, traducciones, explicaciones
- **Como conecta:** Via Google AI Studio
- **Ejemplo:** "Que tengo pendiente en mi vault de Productividad?"

### Copilot
- **Uso:** Asistencia en codigo, documentacion tecnica
- **Como conecta:** Via VS Code o GitHub
- **Ejemplo:** "Documenta este snippet de Python"

## Aplicaciones

### Obsidian
- **Rol:** Editor principal
- **Integracion:** Vault directo
- **Plugins recomendados:** Templater, Dataview, Git

### GitHub
- **Rol:** Control de versiones y backup
- **Integracion:** GitHub Actions para auto-update
- **Funcion:** Sincronizacion automatica diaria

### VS Code
- **Rol:** Editor de codigo y scripts
- **Integracion:** Edicion de archivos .md y .ps1
- **Extensiones:** Markdown All in One, PowerShell

### Notion
- **Rol:** Exportar contenido
- **Integracion:** Exportar vaults a Notion
- **Uso:** Compartir con equipos que no usan Obsidian

### Slack / Discord
- **Rol:** Notificaciones
- **Integracion:** Webhooks para alerts
- **Uso:** Notificar cuando hay cambios en vaults

### Telegram
- **Rol:** Push alerts
- **Integracion:** Bot de Telegram
- **Uso:** Recibir notificaciones en el movil

### Cloudflare
- **Rol:** Deploy de la landing
- **Integracion:** Cloudflare Pages
- **Uso:** Hosting estatico rapido y seguro

## Extensiones Obsidian

### Templater
- **Funcion:** Crear templates dinamicos con JavaScript
- **Ejemplo:** Templates con fecha automatica, variables, funciones
- **Instalar:** Settings > Community Plugins > Templater

### Dataview
- **Funcion:** Queries SQL para tus notas
- **Ejemplo:** "Lista todas las notas creadas esta semana"
- **Instalar:** Settings > Community Plugins > Dataview

### Git
- **Funcion:** Sync directo desde Obsidian
- **Ejemplo:** Commit y push automatico al guardar
- **Instalar:** Settings > Community Plugins > Git

### Calendar
- **Funcion:** Vista de calendario para notas diarias
- **Ejemplo:** Navegar rapido entre diarios
- **Instalar:** Settings > Community Plugins > Calendar

### Periodic Notes
- **Funcion:** Notas diarias, semanales, mensuales
- **Ejemplo:** Estructura automatica para cada periodo
- **Instalar:** Settings > Community Plugins > Periodic Notes

### Excalidraw
- **Funcion:** Diagramas y sketchnotes
- **Ejemplo:** Mapas mentales, flowcharts, wireframes
- **Instalar:** Settings > Community Plugins > Excalidraw
